# ATQT CRM 啟動教學

## 每次開啟網站的步驟

### 第一步：開啟 VS Code 終端機
`Ctrl + `` ` ``（反引號）開啟終端機

---

### 第二步：啟動後端 + 前端（終端機 1）

```powershell
cd "c:\Users\Neil\Desktop\ATQT_CRM\atqt-crm"
npm run dev:all
```

等待看到以下訊息代表成功：
- `server` → `Server running on port 3001`
- `vite` → `Local: http://localhost:5173/`

---

### 第三步：對外分享網址（終端機 2）

按 `+` 新開一個終端機，輸入：

```powershell
cd "c:\Users\Neil\Desktop\ATQT_CRM\atqt-crm"
npm run share
```

等待出現 `Forwarding` 那行，複製 `https://xxxxx.ngrok-free.dev` 的網址分享給團隊。

---

## 本機存取（不需 ngrok）

| 用途 | 網址 |
|------|------|
| 前端 CRM | http://localhost:5173 |
| 後端 API | http://localhost:3001 |
| Ngrok 監控 | http://127.0.0.1:4040 |

---

## 關閉方式

在各終端機按 `Ctrl + C` 即可停止所有服務。
