## 欄位清單

| # | 欄位名稱 | store key | 資料來源 | 說明 |
|---|---------|-----------|---------|------|
| 1 | UID | `uid` | 🔵 API 自動 | BingX 用戶 ID |
| 2 | 新/舊帳號標籤 | `account_type` | 🔵 API 自動 | `'old'` / `'new'` |
| 3 | 直接/間接邀請 | `invite_type` | 🔵 API 自動 | API `directInvitation` → `'直接'` / `'間接'` |
| 4a | 邀請人 UID | `inviter_uid_code` | 🔵 API 自動 | API `inviterSid` |
| 4b | 已綁定邀請碼 | `bound_invite_code` | 🔵 API 自動 | API `inviteCode` |
| 4c | 他的推廣邀請碼 | `own_promo_code` | 🔵 API 自動 | API `ownInviteCode` |
| 5 | 邀請人 LINE 暱稱 | `inviter_line_name` | ✏️ 手動 | 需手動對照填寫 |
| 6a | LINE 稱呼 | `line_name` | 🟢 LINE 匯入 / 📊 XLSX | LINE 對話紀錄或 LINE 名稱對照表 |
| 6b | LINE 顯示名稱 | `line_display_name` | 📊 XLSX | LINE 名稱對照表「您的LINE名字」欄 |
| 7 | 註冊日期 | `register_date` | 🔵 API 自動 | 格式 zh-TW 顯示用 |
| 8 | 近30天交易量 | `volume_recent` | 🔵 API 自動 | 由傭金明細 API 計算（近35天） |
| 9 | 總資產 | `total_assets` | 🔵 API 自動 | API `balanceVolume`（代理商帳號無子帳號資產 API 權限）|
| 10 | 最後交易日 | `last_trade_date` | 🔵 API 自動 | 傭金明細最後有交易的日期 |
| 11 | 官網信箱 | `official_email` | 📊 XLSX | USER-INFO 或 account-tv 表 |
| 12 | TradingView 帳號 | `tradingview_account` | 📊 XLSX | account-tv 表 |
| 13 | BingX 註冊日期 | `bingx_register_date` | 🔵 API 自動 | API `registerDateTime` → YYYY-MM-DD |
| 14 | RFM 標籤 | `tags` / `rfm_score` | ⚙️ 系統計算 | 每次同步後自動重算 |
| 15 | 備註標籤 | `note_tags` | ✏️ 手動 / 📊 XLSX | 陣列，可多標籤；USER-INFO REMARK 欄自動轉入 |
| 16 | 文字備註 | `text_notes` | ✏️ 手動 | 純文字，客戶詳細頁編輯 |
| 17 | 指標版本 | `indicator_version` | ✏️ 手動 | |
| 18 | 社群互動度 | `community_interaction` | ✏️ 手動 | 後續可由 LINE 訊息數自動推算 |
| 19 | BingX VIP 等級 | `bingx_vip_level` | 🔵 API 自動 | API `userLevel` |
| 20 | 首次入金時間 | `first_deposit_time` | 📊 XLSX | USER-INFO「首充時間」欄；API 僅有 `deposit: bool` |
| — | 帳戶餘額 | `balance` | 🔵 API 自動 | 同 `total_assets`，用於 RFM M 分計算 |
| — | 有無入金 | `has_deposit` | 🔵 API 自動 | API `deposit: bool` |
| — | 有無交易 | `has_traded` | 🔵 API 自動 | API `trade: bool` |
| — | KYC 驗證 | `kyc` | 🔵 API 自動 | API `kycResult: bool` |
| — | 近30天交易次數 | `trade_count_30d` | 🔵 API 自動 | 傭金明細記錄筆數 |
| — | 近7天訊息數 | `line_msg_count_7d` | 🟢 LINE 匯入 | 用於 RFM F 分 |
| — | 虛擬活躍日 | `last_active_date` | ⚙️ 系統計算 | 每週匯入/同步時更新心跳，R 分主要依據 |

## 資料來源說明

| 符號 | 來源 | 觸發方式 |
|------|------|---------|
| 🔵 API 自動 | BingX 代理商 API | 點擊「同步資料」下拉選單 |
| 📊 XLSX | BingX 後台匯出 / 自製對照表 | 點擊「匯入 XLSX」 |
| 🟢 LINE 匯入 | LINE 聊天室匯出 `.txt` | 點擊「匯入 LINE 對話」|
| ✏️ 手動 | 人工填寫 | 客戶詳細頁編輯 |
| ⚙️ 系統計算 | 由其他欄位計算 | 每次同步/匯入後自動執行 |
